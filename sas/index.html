Other Sites and Systems

There are a variety of other sites and systems that I either operate or am otherwise associated with that really do not fit elsewhere into this site but I would still like to say something about.
  • Home Server: This is the most powerful server that I own and operate from the small server room in my basement. This is system is used for a variety of both personal and work related tasks such as hosting this website, running VMs for freelance work projects, hardware simulation and software development, as well as file sharing and storage. While uptime issues are very rare, I do avoid hosting anything on this machine where a day or so of downtime is completely unacceptable.
  • IIT Server: This is a much older Dell Poweredge 2850 and IBM EXP400 setup that I own and operate in the Illinois Tech Robotics server center for almost anything that will be primarily accessed from within IIT or other universities such as my coursework repositories, ITR photo/video repositories as well as my Arch Linux mirror.
  • Arch Linux Mirror: This is a full mirror of the Arch Linux distrubution on my server hosted at IIT.
  • Github Page: My page on Github, used for all of my open source hardware and software development. However, since Github's last major outage, I always make sure to keep backups of all repos on one of my own machines.
